Instruction:
请窗口化游玩，程序会自动缩放到一定大小，若定位不准确，请重开游戏

根据你的店铺升级点击每个选项到对应等级，黑色无升级或购买，米色为1级或购买一次，蓝色为2级或购买两次，棕色为3级或购买三次
按键：
S:（培训前）补充配料， （培训后）补充所有
C：切肉
F：切土豆并在土豆炸好后取出
W:做一个标准沙威玛
G：自动提供橙汁和可乐给需要的顾客并重新装填
O：若有橙汁，则自动提供给需要的顾客，没有就先补货
K：若有克比，则自动提供给需要的顾客，没有就先补货
/：若有薯条，则自动提供给需要的顾客，没有就先补货
R：重置橙汁、克比、薯条的计数器防止异常
M：收取所有钱
特殊配料要求的沙威玛：
1：不要酸黄瓜
2：不要酸奶油
3：不要薯条
4：不要石榴糖浆
5：不要酸黄瓜+石榴糖浆（1+4）
6：不要酸奶油+石榴糖浆（2+4）
7：不要薯条+石榴糖浆（3+4）
8：不要酸黄瓜+酸奶油
9：不要酸黄瓜+薯条
0：不要酸奶油+薯条
